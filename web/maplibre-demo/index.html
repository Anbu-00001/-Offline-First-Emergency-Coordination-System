Source: vector (tileserver)

Developer Info

Vector source: Tileserver-gl at localhost:8080. Requires MBTiles in docker/tileserver/data/.

Raster fallback: OpenStreetMap tile CDN (online only).

Public vector fallback: If tileserver is not running, uses demotiles.maplibre.org.

India bounds: 6.55°N–35.67°N, 68.11°E–97.40°E. Zoom 5.

Usage:

1. bash scripts/start_tileserver.sh

2. Open this page: python3 -m http.server 3000

3. Toggle Vector/Raster to compare sources.
